# PyMIO简易爬虫
本项目基于[PyMIO](https://github.com/0ldm0s/pymio)构建，支持环境为MacOS,FreeBSD,Linux和Windows（推荐使用wsl）。
## 文档版本

| 版本号 | 更新日期 |
|:--|:--|
| 0.1 | 2022-07-24 |

# 运行环境
Python 3.7+（windows建议使用3.7.5，不推荐使用嵌入版本）

MongoDB 5.x

FireFox-esr
# 功能目标
- [x] 抓取渲染后的页面源码
- [x] 支持模拟屏（Linux）
- [x] 支持无头模式
- [x] 支持手机模式
- [ ] 多线程
- [ ] 生产者/消费者模式
- [x] 生成树状结构站点
- [x] 下载css/js/img文件
